-Para compilar el programa solo debe colocarse el comando por consola "make", mientras se está en la carpeta "Programa2".

-Se generará un ejecutable llamado "programa2", para utilizar dicho programa se deberá colocar el siguiente comando.

-"./programa2", para abrir los archivos estos deben ser dados como argumentos a la hora de ejecutar el programa por consola de la siguiente manera.

-"./programa2 palabras3 palabras2".

- Los archivos adjuntos para hacer la prueba del programa son los llamados "palabras2" y "palabras3".
